## 📌 Descrição

Este repositório contém a implementação de um programa em Java para resolver equações do segundo grau utilizando Programação Orientada a Objetos (POO).

O programa solicita os coeficientes A, B e C da equação, calcula delta, as raízes X' e X'', e as coordenadas do vértice da parábola, organizando entrada e saída em classes específicas para maior modularidade.

---

🛠 Recursos Utilizados

Linguagem: Java

IDE: NetBeans

Controle de versão: Git

Plataforma de hospedagem: GitHub

---

## 📋 Estrutura do Código

O programa foi organizado seguindo os seguintes critérios:

Dados de entrada e saída são encapsulados em classes específicas (DadosEntrada e DadosSaida).

A lógica de cálculo da equação do segundo grau é implementada em uma classe de serviços (CalcularEqSeqGau2).

O método main realiza a interação com o usuário e exibe os resultados.

Não foram utilizadas variáveis globais; toda a manipulação de dados ocorre via objetos.
